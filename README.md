## Monitoring (Bonus)

Basic monitoring can be implemented using tools like Prometheus and Grafana.

Key metrics to monitor:
- Application uptime
- CPU and memory usage
- Request/response times
- Error rates

Logs can also be monitored using tools like ELK Stack or cloud-based logging solutions.

This setup ensures visibility into application performance and system health.
